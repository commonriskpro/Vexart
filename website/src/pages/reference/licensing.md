# Licensing

Vexart uses a source-available model.

## Free use

- personal projects
- OSI-licensed open-source projects
- commercial entities below `$1M USD ARR`

## Commercial use

Entities at or above `$1M USD ARR` need a commercial license.

- Standard: `$299 / developer / year`
- Enterprise: `$10,000 / year`
