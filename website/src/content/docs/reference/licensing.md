## License Model

Vexart is **source-available** under a dual license:

### Community License (Free)

- Personal projects (non-commercial)
- Open-source projects (MIT, Apache 2.0, BSD)
- Commercial use by entities with **< $1M USD** annual revenue

### Commercial License ($299/dev/year)

Required for entities with **≥ $1M USD** annual revenue.

Includes:
- Email support (48h response SLA)
- Security patches
- Minor version updates

### Enterprise License (Custom pricing)

Starting at $10,000 USD/year.

Includes:
- Priority support (24h SLA)
- Custom theming
- Consulting hours
- Roadmap influence

## Distribution

- Source code is public on GitHub
- Binaries distributed via npm (pre-compiled per platform)
- License verification is honor-based (no DRM)

## FAQ

**Can I use Vexart for free in my startup?**
Yes, if your annual revenue is under $1M USD.

**Do I need a license for open-source projects?**
No, open-source projects with permissive licenses are always free.

**What happens when I cross $1M ARR?**
Purchase a commercial license ($299/dev/year) or contact us for enterprise pricing.
